# Next Project (Starter)

This folder is a starter scaffold for the next software project.

Structure:

- `src/` - application source code
- `tests/` - unit tests (pytest)
- `docs/` - project documentation
- `requirements.txt` - Python dependencies

Quick start

```pwsh
cd next_project
.venv\Scripts\activate    # create/activate a virtualenv if desired
pip install -r requirements.txt
python -m src.main         # run starter script
```
